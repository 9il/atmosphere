@property void  components(Kernels)(Kernels kernels, in T[] grid) if (isInputRange!Kernels && hasLength!Kernels);

Compute internal  components using kernels and grid. The pseudocode of this mathod:  components[i, j] = kernel[i](sample[j]).

Parameters
Kernels kernels Input range of kernels. kernels length equals k. Element type of kernels can be a function pointer, delegate, struct or class with opCall method. Kernel compute feature for each point of grid. In terms of likelihood maximization each kernel is a probability density function.
T[] grid pointers of  components. In terms of likelihood maximization grid is just a sample of length n.
