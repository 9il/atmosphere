Statistical package

    Features

    1. Separating mixtures of probability distributions
    2. Grid methods
    3. Parameterized algorithms
    4. Optimization over sliding window

    Documentation

    Documentation can be found here.

    Instalation

    To use this package, put the following dependency into your project's dub.json into the dependencies section:

    {
    ...
    "dependencies": {
        "atmosphere_gm": ">=0.0.0"
    }
}

    Benchmarking

    It is suggested llvm D compiler be used for benchmarks. Project requires LDC version >= 0.15.0 or DMD >= 2.066, or corresponding GDC release. DMD is easy way to start.
