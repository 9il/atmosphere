Features

  1. Separating mixtures of probability distributions
  2. Grid methods
  3. Parameterized algorithms
  4. Optimization over sliding window

Documentation

Documentation can be found here.

Installation

To use this package, put the following dependency into your project's dub.json into the dependencies section:

{
    ...
    "dependencies": {
        "atmosphere_gm": ">=0.0.2"
    }
}

BLAS

ubuntu

Default package

sudo apt-get install libblas-dev

OpenBLAS

Compile and install optimized OpenBLAS.

OS X

Mac OS X comes with the Accelerate framework built in.

Windows (needs to be checked)

x86

  1. Install latest Win32 BLAS from http://sourceforge.net/projects/openblas/files/.
  2. Copy lib\libopenblas.a to YourProject\blas.lib.

x86_64

  1. Install latest Win64-Int32 BLAS from http://sourceforge.net/projects/openblas/files/.
  2. Copy lib\libopenblas.a to YourProject\blas.lib.
  3. Use dub --arch=x86_64 to run project.

Benchmarking

It is suggested the llvm D compiler be used for benchmarks. Project requires LDC version >= 0.15.0 or DMD >= 2.066, or corresponding GDC release. The DMD is easy way to start.
