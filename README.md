atmosphere
=============
Statistical package.

Testing in progress.

# Numeric methods


1. Solvs optimization problem f(p) -> min, 
where p is discrete probability distribution with k elements.
f = u(Wp),
W - matrix(n rows, k columns),
u - convex function.
2. Separating normal variance mean mixture.
